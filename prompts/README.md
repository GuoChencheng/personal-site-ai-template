# Curated Workflow Prompts

This folder keeps a small public set of prompts that explain useful maintenance directions for the template.

Included:

- `github_public_clean_pack_prompt.txt`: the prompt used to create this public-safe export pack
- `post_access_control_next_steps_prompt.txt`: recommended follow-up work for production-safe editor access
- `post_mvp_next_steps_prompt.txt`: broader post-MVP improvement ideas

Older historical build prompts were intentionally excluded from the public pack to keep the repository focused.
